The reduced Meerkat binary surfaces have been retired. The repository no longer
builds or publishes separate reduced CLI or JSON-RPC binaries.

## Current surfaces

Use the public runtime surfaces instead:

| Surface | Purpose |
|---------|---------|
| `rkat` | Batteries-included terminal UX |
| `rkat-rpc` | JSON-RPC surface used by the Python and TypeScript SDKs |
| `rkat-rest` | REST surface |
| `rkat-mcp` | MCP server surface |

## Custom distributions

If you need a smaller product surface for embedding or custom packaging, start
from the shared runtime and surface composition layers rather than reviving the
old reduced binaries. The current codebase keeps feature-gated capabilities in
the main surfaces and no longer carries dedicated mini binary targets or a
separate reduced RPC method catalog.
