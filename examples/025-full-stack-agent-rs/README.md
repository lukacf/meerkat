# 025 вҖ” Full-Stack Agent (Rust)

The reference architecture for production Meerkat agents. Combines every
feature: custom tools, built-in tools, skills, budget control, session
persistence, event streaming, and composable dispatchers.

## Features Used
- `AgentBuilder` with full configuration
- `CompositeDispatcher` вҖ” merge built-in + domain tools
- `BudgetLimits` вҖ” production cost control
- Skills вҖ” injected behavioral instructions
- Event streaming вҖ” real-time monitoring
- Session persistence вҖ” survives restarts
